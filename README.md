A portfolio of data cleaning and exploratory data analysis (EDA) projects using Python, pandas, NumPy, Matplotlib, and Seaborn — transforming raw datasets into clean, insightful, and ML-ready data.

> Exploratory Data Analysis (EDA) of the Titanic dataset using Python, pandas, NumPy, Matplotlib, and Seaborn — includes data cleaning, handling missing values, visualizing survival patterns, and
> preparing the dataset for machine learning.
